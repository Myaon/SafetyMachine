# is_color_alert.py
特定の色の有無で正常異常を判断します。

異常時は物理スイッチの状態をもとに  
・アラート音  
・IFTTT経由でのスマートプラグOFFもしくはSwitchBot経由でスイッチ付きプラグOFF  
ができるようになっています。

## 動作確認済み環境
- Raspberry Pi 3B
- Raspbian OS
